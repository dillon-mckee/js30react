# JavaScript30-React

WIP JavaScript 30 Day Challenge in React

For more info about the Javascript 30 day challenge: https://javascript30.com

## Setup

### Clone and NPM Install

Replace `<project name>` with whatever want to call your project

``` bash
git clone https://github.com/dillon-mckee/javascript30react.git <project name>
cd <project name>
npm install
```

### Build and Dev

Build: run a single build

``` bash
npm run build
```

Watch: watches directory and automatically rebuilds

``` bash
npm run watch
```

Serve: starts http server on port 8080

``` bash
npm run start
```

Dev: runs watch and start simultaneously

``` bash
npm run dev
```
